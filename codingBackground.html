My favourite languages ranked

1. C
C is my favourite language, it is the first language I learned that made me appreciate low-level programming. It taught me about memory management, pointers, and how computers work at a low level. I love how simple it is, yet so powerful. I can do so much with it, from making games to operating systems. It is a language that will always have a special place in my heart. At first I didn't like it since I was coming from C#, but it really grew on me, it became my go to language, and now, I've chosen to ditch unity completely and make a game using C and SDL2 for my CS50 final project. I'm having a great time with it, and I hope to continue using C for a long time.
2. C#
C# is my second favourite language, it is the first language I learned and it will always have a special place in my heart. It is a language that is easy to learn, yet so powerful. I love how it has so many libraries and frameworks that make development so much easier. I used to use it for game development, but now I want to use it for web development and windows apps creation. It is a language that I will always come back to, especially with so much Brackeys content to consume, it'll never get old to me. Coming back to C# from unity was a bit weird honestly, but that made me like it more since it abstracted so many things I had to code by hand in C (I'm looking at you "strings", or should I say char *). I'll definitely continue using C# for a long time, especially with the new features being added to it. Who knows, maybe I'll publish a game made with C# one day, like ConcernedApe did with Stardew Valley.
3. Java
Java is a language I learned simply because of one tiny fact, Notch used it to create Minecraft. I know I know, but wow, it's quite the versatile language. I learned it not so long ago and I was surprised by how much it was so similar to C#. I learned it for the sake of learning, and I have to say, it is a great language. Honestly, I'd go as far as to say it is the better version of Python (Spoiler, you'll never see Python make an appearance on this list). I wonder why it isn't taught in schools more, I'd say it's quite the great starting point.
4. C++
Can I start off by saying, C++ is so weirdüòÖ, I mean I know it's the better version of C but wow, it might as well be its own thing. I learned C++ after learning C#, and it was a bit of a shock to me. Back then I didn't know about pointers, or memory management or anything CS50 taught, so I was definitely confused. I muscled through it though, and I learned a lot from it, though I still prefer C of course.
